# 🎮 Java Game Lobby (Java 遊戲大廳)

一個基於 Java 開發的終端機/圖形化（請根據實際情況修改）遊戲大廳系統。玩家可以在此系統中註冊、登入，並遊玩多款經典的街機遊戲，系統同時支援完整的玩家管理、排行榜與成就追蹤功能。

---

## ✨ 核心功能

### 🕹️ 內建經典遊戲
1. **貪吃蛇 (Snake)**：經典的貪吃蛇遊戲，考驗玩家的反應與走位。(已實裝)
2. **經典 2048**：透過滑動方塊進行數字合併，挑戰達成 2048 的極限。(未實裝)
3. **俄羅斯方塊 (Tetris)**：經典的方塊消除遊戲，支援旋轉、加速下落與得分計算。(未實裝)

### 👥 帳號與玩家管理 (CRUD)
* **安全登入/登出**：保障玩家的資料安全與遊戲紀錄。
* **玩家 CRUD 功能**：
  * **Create**：新玩家註冊帳號。
  * **Read**：查看玩家個人檔案（含歷史勝率、遊玩時間等）。
  * **Update**：修改密碼、更改玩家暱稱。
  * **Delete**：註銷/刪除玩家帳號。

### 🏆 榮譽系統
* **玩家排行榜 (Leaderboard)**：即時更新每款遊戲的前幾名高分玩家，激發競爭心。
* **成就系統 (Achievements)**：解鎖特殊榮譽（例如：「初試身手」、「2048 大師」、「貪吃蛇千分俱樂部」）。
